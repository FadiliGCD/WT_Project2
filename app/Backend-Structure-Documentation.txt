src/ - Application Source Code: (This folder separates the main logic from configuration files.)
The src folder contains all the core backend source code of the application.
It includes server configuration, database connection, routes, middleware, and data models.

The models folder contains the Mongoose schemas and models used to represent the application's data in MongoDB.
Each model defines the structure of a document stored in the database.
